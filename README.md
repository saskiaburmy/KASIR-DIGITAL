# 🏪 Sejuta Tawa - Point of Sales (POS) System

Aplikasi Kasir Digital modern, bersih, dan estetik yang dirancang khusus untuk memenuhi kebutuhan operasional **Kafe Sejuta Tawa**. Project ini dibuat untuk mempermudah kasir dalam mencatat pesanan pelanggan, menghitung total belanjaan secara otomatis, dan mengelola menu kafe berdasarkan kategori.

## ✨ Fitur Utama Aplikasi
* **Manajemen Menu Terstruktur:** Menu dikelompokkan dengan rapi berdasarkan kategori (Makanan Utama, Dessert, Snack, dan Minuman).
* **Keranjang Belanja Interaktif:** Kasir dapat menambah atau mengurangi jumlah item pesanan dengan mudah.
* **Input Nomor Meja:** Dilengkapi kolom khusus nomor meja untuk memastikan pesanan tidak tertukar.
* **Desain Responsif & Estetik:** Tampilan antarmuka yang bersih, modern, dan sangat nyaman digunakan di berbagai perangkat (Laptop/Tablet).

---

## 🛠️ Alur Pengembangan Project (Workflow)
1. **Perencanaan & Analisis:** Menentukan kebutuhan menu dan alur transaksi kasir Kafe Sejuta Tawa.
2. **Desain Antarmuka (UI):** Menyusun tata letak aplikasi kasir agar ramah pengguna (*user-friendly*).
3. **Integrasi Sistem:** Membangun logika perhitungan total harga, pajak, dan manajemen *cart* belanja via platform Base44.
4. **Publish & Dokumentasi:** Mengonlinekan aplikasi secara *live* dan mengelola kolaborasi tim menggunakan Git & GitHub.
