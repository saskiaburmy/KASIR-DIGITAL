# Sejuta Tawa - Point of Sales (POS) System

A modern and aesthetic Point of Sales (POS) application designed for "Sejuta Tawa" Cafe. This app helps cashiers manage orders, calculate totals, and view available menus efficiently.

## 🔗 Live Demo
You can try the live application here: [Launch Sejuta Tawa POS](https://sejutatawa-pos-flow.base44.app)

## ✨ Features
- **Menu Management:** Organized by categories (Makanan, Dessert, Snack, Minuman).
- **Interactive Cart:** Easily add items and input table numbers.
- **Responsive Layout:** Clean and user-friendly interface.
